# OpenRateVC

OpenRateVC 是一个面向创业者的投资人筛选工具，目标是帮助创始人在有限时间内更快地从公开信息中筛选潜在匹配的 VC、基金和投资人。

## 项目理念

- **方便创业者快速筛选**：把投资人、机构、地区、关注阶段、投资偏好和评分信号整理成移动端友好的浏览体验，帮助创业者先做第一轮高效筛选。
- **公开可查数据**：项目只基于公网可查信息、公开页面、公开资料和可核验线索整理，不使用私密数据或非公开来源。
- **先筛选，再尽调**：OpenRateVC 不是投资建议，也不替代创业者自己的判断。它更适合作为找投资人、排优先级、准备沟通名单的辅助工具。
